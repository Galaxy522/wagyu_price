# Git Lesson


## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？
- リモートリポジトリ：リモート上の保管場所
- ローカルリポジトリ：PC上の保管場所

## プッシュとマージの違いは何でしょうか？
- プッシュ：リモートリポジトリにデータを取り込むこと
- マージ：ローカルブランチ同士、リモートブランチ同士それぞれでデータを取り込むこと

## コミットとプッシュの違い
- コミット：プッシュの前状態でステージングに上げること
- プッシュ：リモートリポジトリにデータを取り込むこと

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
- 変更した点をメモに残しておくと分かりやすい

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
- プルリクエストでマージするフローの場合、レビュー者が最終的にマスターブランチにマージするためマージ前に確認ができる。ローカルでマージするフローの場合、自身でマスターブランチにプッシュするため確認者が介入せずリスクの恐れがある

## コンフリクトを起こしてしまった場合、どう対処すべきですか？
- 対処法は以下３つあり、状況次第で選択する
  - 先にマージされた変更内容を取り込む
  - 後にマージしようとしている変更内容を取り込む
  - どちらの変更内容も取り込む
